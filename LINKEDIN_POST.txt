🚀 TRY IT YOURSELF
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Clone the repository:
   git clone https://github.com/j1c4b/retail-bank-sim.git
   cd retail-bank-sim

2. Run automated setup:
   ./setup.sh

3. Test streaming pipeline:
   python transaction_generator.py  # Terminal 1
   python bq_consumer_dlp.py        # Terminal 2

4. Or test batch pipeline (production):
   python batch_generator_avro_dlp.py

5. Clean up resources:
   ./cleanup.sh
